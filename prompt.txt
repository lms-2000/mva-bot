Создать telegram бота на ts в этой папке
у бота должна быть кнопка показывающая список доступных команд start send
конманда start запускает интерактивное меню "Главный корпус" "УЛК" "Анатомия" "ДОС2" "Общежитие-1" "Общежитие-3" "Общежитие-4" "Общежитие-5" "Общежитие-6" "Общежитие-7"
пользователь выбирает любой пункт меню 
выводится сообщение "Подробно оптшите Вашу проблему, прикрепите фотографии, видео. В меню выберите команду /send"
пользователь вводит текст, может прикрепить фотографии, видео или любой другой документ
по команде /send отчет администратору, настройки администратора хрянтся в файле конфигурации
